Interactive Search Experiment Example


Welcome!


This website allows you to complete an example interactive search task and download your associated data.

Use the drop down list to select the number of trials you want to complete and click the button to start the experiment.

You will be shown an instruction page before you start.



Note on Data Collection

We do not store any of your data. As soon as you close the webpage your data is destroyed. Only you will have access to your data if you choose to download it at the end of the experiment.
